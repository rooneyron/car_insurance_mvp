MVP 核心工作（面试精简版）
架构设计与工具解耦
设计手写 StateGraph 4 节点编排（router → planner ⇄ tools → responder），将工具层与 Agent 解耦，支持工具短路直返机制，为后续 MCP 迁移预留扩展接口。

RAG 检索系统
实现了基于 FAISS + FastEmbed 的本地向量检索，并通过 Rerank 策略（本地 Cross-Encoder / 线上 LLM 重排）提升了检索精度。

跨轮记忆与上下文管理
集成了 Memory 摘要压缩机制，当对话超过 token 阈值时自动生成摘要，解决了多轮对话上下文膨胀的问题。

生产环境内存优化
将 sentence-transformers 替换为 fastembed，使应用在 Render 512MB 免费实例上从无法启动变为稳定运行（内存从 800MB 降至 300MB）。

公网安全与成本控制
实现了 JWT Token 访问控制（7 天有效期）和每日 Token 限额管理，确保公网演示的安全性和成本可控。

全链路可观测性
添加了性能计时、Token 消耗 JSON 日志和前端耗时显示，便于线上问题排查和成本分析。

完整部署上线
项目已部署至 Render 公网，可通过链接直接访问演示，支持冷启动自动唤醒。
